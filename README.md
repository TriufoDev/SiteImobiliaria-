# Imobiliária Santos - Website

Site desenvolvido para a Imobiliária Santos com HTML e CSS.

## Estrutura do Projeto

```
imobiliaria/
├── index.html          # Página principal
├── style.css           # Estilos do site
├── assets/             # Pasta de imagens
│   ├── logo.png        # Logo da imobiliária (criado)
│   ├── telefone.png    # Ícone de telefone (criado)
│   ├── banner.jpg      # Imagem do banner principal (ADICIONAR)
│   ├── casa.png        # Imagem da casa modelo (ADICIONAR)
│   └── corretores.jpg  # Imagem dos corretores (ADICIONAR)
└── textos/
    └── textos-da-pagina.txt
```

## Imagens que Você Precisa Adicionar

O site já está funcional, mas você precisa adicionar as seguintes imagens na pasta `assets/`:

1. **banner.jpg** - Uma foto bonita de uma casa/imóvel para o banner principal
   - Recomendado: 1200x500px
   - Pode usar a imagem que você me enviou do anexo

2. **casa.png** - Uma ilustração ou foto de uma casa para a seção "Sobre"
   - Recomendado: 300x300px ou similar
   - Pode ser uma casa 3D ou ilustração

3. **corretores.jpg** - Uma foto de corretores ou profissionais em reunião
   - Recomendado: 350x250px
   - Imagem profissional de negócios

## Como Usar

1. Adicione as imagens mencionadas acima na pasta `assets/`
2. Abra o arquivo `index.html` no navegador
3. O site está totalmente responsivo e pronto para uso!

## Características

- ✅ Design responsivo (funciona em mobile e desktop)
- ✅ Navegação suave entre seções
- ✅ Mapa integrado do Google Maps
- ✅ Seções: Sobre, Serviços, Corretores e Contato
- ✅ Estilo moderno com cores azul e branco
- ✅ Logo e ícones incluídos

## Cores Utilizadas

- Azul claro (header): #87CEEB
- Azul primário: #1E90FF
- Cinza: #333, #555, #666
- Branco: #FFF
- Fundo cinza claro: #F5F5F5
